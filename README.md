# 数据结构与算法练习

个人数据结构与算法学习仓库，包含基础数据结构实现和常见面试题解答。

## 📁 目录结构

```
.
├── src/                    # 基础数据结构实现
│   ├── 线性表/            # 数组、链表、栈、队列
│   ├── 树/                # 二叉树、BST、AVL、红黑树
│   ├── 图/                # 图的表示与遍历
│   └── 排序/              # 各种排序算法
├── interview/             # 面试题整理
│   ├── 岛屿问题/          # 并查集、DFS、BFS
│   └── 会议室安排/        # 贪心算法
├── docs/                  # 学习笔记与文档
└── tests/                 # 测试代码
```

## 🚀 快速开始

所有代码使用 C 语言编写，编译运行：

```bash
gcc -o output filename.c
./output
```
